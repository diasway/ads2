===== Implemented Data Structures

- **MyArrayList<T>** – Custom dynamic array implementation
- **MyLinkedList<T>** – Doubly linked list with indexed access
- **MyMinHeap<T>** – Generic Min Heap implementation
- **MyStack<T>** – Stack (LIFO) based on dynamic array
- **MyQueue<T>** – Queue (FIFO) based on linked list

All classes use a shared interface: `MyList<T>`, which defines the list operations.
